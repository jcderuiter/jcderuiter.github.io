De letters HSL staan voor Hue (tint), Saturation (verzadiging) en Lightness of Luminosity (helderheid).
    De hue is verdeeld in 360°, saturation en lightness worden in een percentage uitgedrukt.
    De hue bepaalt de kleur, als het ware een cirkel doorlopend van rood (0°) via groen (120°) en blauw (240°) weer naar rood (360°); saturation loopt van grijs (0%) tot de volledig verzadigde kleur (100%), en lightness loopt op van donker (0% is zwart) naar licht (100% is wit). Een lightness van 50% is de "normale" helderheid.
    Bovenstaand schema geeft de variatie in saturation (van links naar rechts oplopend) en lightness (van beneden naar boven oplopend) in stappen van 10% voor een bepaalde hue.
    Hieronder hue waarden oplopend in stappen van 15, alle met 100% saturation en 50% lightness; als je er op klikt wordt de hue in het schema hierboven toegepast.

    CSS, JavaScript

    In CSS bestaat de functie hsl(hue, sat, lum), zoals er ook een rgb(red, green, blue) is.
    De rgb() functie gebruikt een systeem, RGB (Red Green Blue), waarin rood, groen en blauw elk een "gewicht" van 0 tot en met 255 kan hebben.

    Je kunt een HTML element een kleur (of achtergrondkleur) geven met de hsl() functie in CSS:
    background-color: hsl(240, 100%, 50%);
    In de hsl() functie geef je de hue als alleen een getal (of als angle unit) op, maar de saturation en lightness moet als percentage opgeven. In de hsl() functie mogen decimalen staan: hsl(177.81, 59.83%, 55.10%), in de rgb() functie gebruik je alleen gehele getallen.
    In JavaScript codeer je de functies als string, tussen quotes:
    element.style.backgroundColor = "hsl(240, 100%, 50%)";
    Als je in JavaScript een de kleur weer opvraagt, bijvoorbeeld met:
    var clr = getComputedStyle(element).backgroundColor;
    dan krijg je de rgb() functie terug, als string: clr krijgt waarde "rgb(0, 0, 255)".

    RGB naar HSL

    Bij de vertaling van RGB naar HSL wordt gekeken naar de maximale en minimale bijdrage van rood, groen en blauw: 
    Cmax = max(red, green, blue) / 255
Cmin = min(red, green, blue) / 255
    De lightness wordt gedefinieerd als 
    L = (Cmax + Cmin) / 2
    Een lightness van 100% (L = 1) treedt op bij rgb(255, 255, 255) ofwel wit; en 0% bij rgb(0, 0, 0) ofwel zwart.
    Definieer λ = 1 - |2L - 1|. Dus λ is minimaal 0 als L = 0 of L = 1 en maximaal 1 als L = 0.5.
    Definieer Δ = Cmax - Cmin.
    De saturation wordt gedefinieerd als 
    λ = 0 : S = 0
λ ≠ 0 : S = Δ / λ
    Een 0% saturation treedt dus op als Δ = 0 ofwel als de hoeveelheden rood, groen en blauw even groot zijn. (Of bij λ = 0 ⇔ L = 0 ∨ L = 1 ofwel wit of zwart.)
    S uitgedrukt in Cmax en Cmin: (Cmax - Cmin) / (Cmax + Cmin) als Cmax + Cmin < 1 of (Cmax - Cmin) / (2 - (Cmax + Cmin)) als Cmax + Cmin > 1.
    Een 100% saturation treedt dus op als Δ = λ (≠ 0) ofwel als Cmin = 0 of als Cmax = 1.

    De hue hangt samen met de mate waarin rood groen en blauw vertegenwoordigd zijn. Definieer geschaalde waarden voor rood, groen en blauw: R = red/255, G = green/255 en B = blue/255.
    De hue wordt gedefinieerd als 
    Δ = 0   : H = 0
Cmax = R : H = 60 ×(G - B) / Δ (mod 360)
Cmax = G : H = 60 ×(B - R) / Δ + 120
Cmax = B : H = 60 ×(R - G) / Δ + 240
    Dus als R de grootste is heb je -60° ≤ H ≤ 60°, als G de grootste is 60° ≤ H ≤ 180° en als B de grootste is 180° ≤ H ≤ 300°.

    HSL naar RGB

    De vertaling van HSL naar RGB is uiteraard het omgekeerde van bovenstaande.
    Waar we naar toe werken is zoiets als (red, green, blue) = (R × 255, G × 255, B × 255), met 0 ≤ R,G,B ≤ 1. Deze R, G en B zijn een permutatie van Cmax, Cmin en een waarde daar tussenin, C2.
    Uit de saturation en de lightness kunnen Cmin en Cmax berekend worden. 
    Cmax + Cmin     = 2L                      Cmin = L - ½λS
Cmax - Cmin = Δ = λS                      Cmax = L + ½λS
    De waarde van de laatste kleur, C2, en hoe Cmin, C2 en Cmax aan R, G en B worden gekoppeld moet uit de hue volgen.
    De definitie van de hue zegt: H = 60 × (X - Y) / Δ + α (α = 0, 120, 240, 360).
    Definieer (H / 60) %2 = γ, dit is een getal tussen 0 en 2; als γ ≤ 1 dan was X = C2, Y = Cmin en als γ > 1 dan was X = Cmin, Y = C2.
    Dus als γ ≤ 1 dan is (C2 - Cmin) / Δ = γ
    En als γ > 1 dan is (Cmin - C2) / Δ + 2 = γ
    Definieer η = 1 - |γ - 1|
    Dan krijg je
    C2 = ηΔ + Cmin = L + (η - ½)λS

    Bijvoorbeeld hsl(165, 80%, 60%): 
    λ = 1 - |2×0.6 - 1| = 0.8
η = 1 - |(165/60) % 2 - 1| = 1 - |2.75 % 2 - 1| = 1 - |0.75 - 1| = 0.75
Cmax = L + ½λS       = 0.6 + 0.5 × 0.8 × 0.8          = 0.92
Cmin = L - ½λS       = 0.6 - 0.5 × 0.8 × 0.8          = 0.28
C2   = L + (η - ½)λS = 0.6 + (0.75 - 0.5) × 0.8 × 0.8 = 0.76
    En omdat 120 ≤ H < 180 is (red, green, blue) = (Cmin × 255, Cmax × 255, C2 × 255) = (71, 235, 194)

    Je ziet dat Cmax, Cmin en C2 alle drie de vorm L + βS hebben; begrijpelijk, want als S = 0 is de HSL een grijstint die alleen van de lightness afhangt.
